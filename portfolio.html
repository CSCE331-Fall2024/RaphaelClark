Making a Programming Language Interpreter

In this project I made something that I consider to be one of the most difficult and entertaining things I have ever created. A MUPL, or a made up programming language is something that uses a current language to make a new language with its own syntax and tokens, but using the foundation of another language. This was Racket for me. In the end, my programming language could recognize all the basic variable types in addition to being able to recognize things like functions, classes, and environments. the language could use higher order functions, and fold, map, and filter things. It was a great endeavor and a difficult project but was very fun and incitefull.
